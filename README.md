# Gestordefinancias

> ele gere seu money 💰

Ferramenta de orçamento pessoal em arquivo único HTML — funciona em qualquer dispositivo, sem instalação.

## Funcionalidades

- 📊 Painel com renda, método (50/30/20, 6 Potes, ou Personalizado) e gráfico de pizza
- 💸 Lançamento de despesas com auto-sugestão de categoria
- 🔁 Despesas recorrentes (aluguel, assinaturas)
- 🎯 Metas de poupança com prazo e progresso
- 📈 Histórico mensal com gráfico de barras comparativo
- 🌙 Tema claro/escuro
- 📱 Otimizado pra mobile (PWA-ready)
- 💾 Backup em JSON
- ↶ Undo ao excluir
- 🔍 Busca e filtros nas despesas

## Como usar

Abra o `index.html` no navegador. Tudo é salvo localmente no `localStorage`.

Para usar em qualquer dispositivo: publique o arquivo (Netlify, GitHub Pages, Vercel) e acesse pelo link.

## Privacidade

100% client-side. Seus dados nunca saem do seu navegador, exceto quando você exporta um backup manualmente.
